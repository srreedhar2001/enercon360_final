📋 How to Use Registration Status Toggle

📱 For Creating New Users:

  1. 1. Navigate to User Management page
  2. 2. Click "Add User" button
  3. 3. Fill in user details
  4. 4. Choose Registration Status:
  5. • Active: User can log in immediately
  6. • Inactive: User account disabled
  7. 5. Click "Add User" to save

✏️ For Editing Existing Users:

  1. 1. Find user in the list
  2. 2. Click "Edit" button on user card
  3. 3. Modal opens with current details
  4. 4. Current status is pre-selected
  5. 5. Change status if needed
  6. 6. Click "Update User" to save changes
